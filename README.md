## Описание

Этот проект демонстрирует конвейер CI/CD с использованием Docker и GitHub Actions.

## Установка

```bash
# Настройка зависимостей
make setup

# Запуск тестов
make test

# Запустить сервер разработчика
make dev

# Запуск с Docker Compose
make docker-dev
